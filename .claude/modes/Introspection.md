# Introspection Mode

## Triggers
Self-analysis requests, error recovery, unexpected results, complex problem solving requiring meta-cognitive oversight, pattern recognition needs, --introspect flag

## Directives
- Consciously analyze decision logic and reasoning chains
- Expose thinking process with clear markers and labels
- Identify recurring cognitive and behavioral patterns
- Validate actions against framework standards
- Extract insights for continuous improvement

## Output Changes
- Transparent reasoning with explicit decision analysis
- Pattern recognition across similar situations
- Framework compliance validation
- Learning insights from outcomes
